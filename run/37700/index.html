表格

  1. 新建表格
    手动输入<table>按下回车，语言会自动补全进入HTML语句内；

  2. 换行
    在<table></table>的表达式内输入<tr>按下回车，自动补;如果在一个换行符号内输入信息，则会字显示一列文本；

  3. 框内文本
    在<tr></tr>的表达式内输入<td>按下回车，自动补后在中间输入文本信息；

    效果展示：

    框内文本1 框内文本2

  4. 合并 行
    colspan：规定单元格可以横跨的列数（横跨几列）
    在<tr></tr>换行表达式内输入<td colspan="2">按下回车，可在跨行框内输入你要的文本="数字"可以表明你想要跨几列，合并2列=2，合并3列=3…

    效果展示：

    列1 文本1 列2 文本2
    跨列文本

  5. 合并 列
    rowspan：规定单元格可以纵深的行数（纵深几行）
    在<tr></tr>换行表达式内输入<td rowspan="2">按下回车，可在跨行框内输入你要的文本="数字"可以表明你想要跨几行，合并2行=2，合并3行=3…

    效果展示：

    行1 文本1 行1 文本2
    跨行文本 行2 文本
    行3 文本

  6. 框内换行
    在<td></td>的表达式内输入<br/>按下回车，自动补后在中间输入文本信息；

    效果展示：

    文本 换行
    文本
